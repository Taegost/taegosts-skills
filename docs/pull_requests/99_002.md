## Groups

### Group A — `validate-index-standards.py` (Findings 1, 3, 4, 5)

All four findings target the same file. No interdependencies between them — each modifies a different function/line.

**Finding 1:** Strip inline code spans (backtick-delimited) from lines before running bare-URL regex checks in `check_r7_links`. Use `re.sub(r'`[^`]+`', '', line)` on a working copy of the line before the URL scan loops. Keep the original line for error messages.

**Finding 3:** Replace `dict | None` with `Optional[dict]` and add `from typing import Optional` to imports.

**Finding 4:** Add `in_code_block` tracking to `extract_links` (lines 46-60), mirroring the pattern from `check_r7_links` (lines 70-77). This prevents false R8 scope violations for links inside code blocks.

**Finding 5:** The bare-URL regex `(?:^|\s)(https?://[^\s]+)` matches URLs after spaces inside parentheses (e.g., `[text]( https://example.com)`). The `before` substring ends at the space, so the `re.search(r'\]\($', before)` check fails. Fix: strip trailing whitespace from `before` before checking for `](`: `before = line[:match.start()].rstrip()`.
