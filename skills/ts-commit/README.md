# ts-commit

Infrastructure skill for creating well-crafted git commits.

## Purpose

This skill provides the `ts-commit` workflow for creating clear, atomic git commits
from the current working tree changes. Use `ts-commit-push-pr` if you also need to
push and open a PR.

## Structure

- `SKILL.md` - Main skill definition and instructions
- `scripts/` - (future) Extracted helper scripts
