# ts-commit-push-pr

Infrastructure skill for committing changes, pushing to remote, and opening a PR.

## Purpose

This skill provides the `ts-commit-push-pr` workflow that combines:
1. Creating a well-crafted git commit
2. Pushing the branch to remote
3. Opening a pull request

Use `ts-commit` if you only need to commit without pushing.
